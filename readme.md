# DX Pass Generator

基于 PIL 的 DX Pass 图片生成器。

> [!WARNING]
> 此项目正在开发当中。
> 目前的可用程度：**不可用**。

## 安装

1. 安装 Python 3.12 或更高版本；
2. 克隆该项目：
   ```
   git clone https://github.com/nhdsd/dx-pass-generator
   ```
3. 安装依赖：
    ```
    pip install -r requirements.txt
    ```
